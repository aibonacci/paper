Composite Wythoff Sequences and Fibonacci Linear Structures

1. Core Structure: Nested Maps and the Golden Ratio

1.1 Basics: \(a(n)=\lfloor n\varphi\rfloor\), \(b(n)=\lfloor n\varphi^2\rfloor\)

Lower and upper Wythoff sequences \(a(n), b(n)\) are closely related to the golden ratio \(\varphi\), satisfying:

\[ b(n)=a(n)+n, \quad \varphi^2=\varphi+1. \]

Composing these functions (e.g., \(a(b(n))\) or \(b(a(n))\)) leverages the algebraic property of \(\varphi\).

1.2 Instruction Strings and Function Nesting

Each composite instruction string \(\ell_1\ell_2\cdots\ell_k\) (with \(\ell_i\in\{1,2\}\)) represents sequentially applying the functions \(a\) or \(b\). Repeated floor operations introduce systematic rounding errors that are captured by linear Fibonacci expressions.

2. Linear Fibonacci Representation: Coefficients and Weights

2.1 Weights of Instruction Strings \(\sum \ell_i\)

The weight \(\sum_{j=1}^{k} \ell_j\) of each instruction string closely determines the Fibonacci coefficients that appear.

2.2 Linear Combination: \(\alpha a(n)+\beta b(n)-\text{offset}\)

Composite functions form structures of the type \(\alpha a(n)+\beta b(n)\), with coefficients \(\alpha,\beta\) typically Fibonacci numbers. Each choice of \(\ell_i\) leads to predictable offsets.

3. Essence of Structure: Repeated Floor Operations and Fibonacci Recurrences

  • Mathematical foundation: Algebraic properties of \(\varphi\).
  • Basic objects: \(a(n)=\lfloor n\varphi\rfloor\), \(b(n)=a(n)+n\).
  • Composition: Nested floor operations generate structured errors expressible via Fibonacci sequences.
  • Fibonacci patterns: Recursive emergence of Fibonacci sequences.
  • Unified offset: Errors accumulated systematically into a single closed-form offset.
