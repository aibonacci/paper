我的新定理

引言

黄金分割率（golden ratio） $\\varphi = \\frac{1+\\sqrt{5}}{2} \\approx 1.618$ 在数论和组合数学中具有重要地位。由$\\varphi$产生的两组数列 $W_1(n) = \\lfloor n\\varphi\\rfloor$ 和 $W_2(n) = \\lfloor n\\varphi^2\\rfloor$ （其中符号$\\lfloor x\\rfloor$表示不大于$x$的最大整数，即高斯符号）是著名的 Wythoff 对偶数列，它们满足 $\\frac{1}{\\varphi} + \\frac{1}{\\varphi^2} = 1$， 因此根据贝亚蒂（Beatty）定理，这两个数列互为补集，所有正整数在其中恰好出现一次 (威佐夫游戏 - 维基百科，自由的百科全书)。 直观地说，$W_1(n)$与$W_2(n)$分别给出了从自然数中挑选出的两个互补子序列。

这些源于黄金分割率的数列还与斐波那契数有密切关联：例如，黄金分割率满足关系 $\\varphi^2 = \\varphi + 1$， 从而诱导出斐波那契递推关系。在本文中，我们关注对数列 $W_1(n)$ 和 $W_2(n)$ 的反复迭代复合。具体而言，定义 复合数列 $W_{l_1l_2\\dots l_k}(n)$ 表示按顺序对$n$依次作用函数$W_{l_k}, W_{l_{k-1}}, \\dots, W_{l_1}$。也就是说，先计算 $W_{l_k}(n)$，然后将结果代入$W_{l_{k-1}}$，如此迭代，最后计算$W_{l_1}$的值。每个下标$l_i$取值为1或2，因此复合数列是由$W_1$和$W_2$这两个基本序列反复嵌套而成的。例如， $W_{12}(n) = W_1(W_2(n))$， $W_{21}(n) = W_2(W_1(n))$， $W_{11}(n) = W_1(W_1(n))$， $W_{22}(n) = W_2(W_2(n))$， 等等。

通过直接计算一些小的例子，可以发现有趣的模式：例如 $W_{12}(n) = W_1(n) + W_2(n)$， $W_{21}(n) = W_1(n) + W_2(n) - 1$， $W_{11}(n) = W_2(n) - 1$， $W_{22}(n) = W_1(n) + 2W_2(n)$ 等等。更进一步，在整数数列数据库 OEIS 的研究中也记录了类似的观察。例如，记 $A=\\{W_1(n)\\}$为下 Wythoff 数列、 $B=\\{W_2(n)\\}$为上 Wythoff 数列，则复合数列$AB$满足 $AB = A + B$，复合数列$BB$满足 $BB = A + 2B$，而$BBB$（三次复合$W_2$）满足 $BBB = 3A + 5B$ (A001950 - OEIS)。这些关系中出现的系数 $1,2,3,5$ 正是斐波那契数列的一部分，暗示了复合数列与斐波那契数之间存在一般性的联系。 本论文的主要结果即为刻画任意长度的此类复合数列与斐波那契数的线性关系。该新定理不仅统一了上述特殊情况，而且提供了一个普遍适用的公式。

背景

我们首先回顾斐波那契数列的定义和性质。斐波那契数列 $\\{F_n\\}_{n\\ge 0}$ 由递推关系 $F_n = F_{n-1} + F_{n-2}$（$n\\ge 2$）定义，初始值为 $F_0 = 0,\\;F_1 = 1$。由此可以推导出若干扩展值，例如为了形式上的方便，我们令 $F_{-1} = 1$（这使得 $F_1 = F_0 + F_{-1}$ 成立）。斐波那契数列的前几项为 $0,\\;1,\\;1,\\;2,\\;3,\\;5,\\;8,\\;13,\\dots$。

黄金分割率 $\\varphi = \\frac{1+\\sqrt{5}}{2}$ 是与斐波那契数密切相关的无理数。它是方程 $x^2 = x + 1$ 的正根，并满足著名的极限关系 $\\lim_{n\\to\\infty}\\frac{F_{n+1}}{F_n} = \\varphi$。黄金分割率还有一个重要性质：对于任意正整数$n$，都有 $\\lfloor n\\varphi \\rfloor + \\lfloor n\\varphi^2 \\rfloor = n\\varphi^2 + n\\varphi - \\{\\! n\\varphi \\!\\} - \\{\\! n\\varphi^2 \\!\\} = n\\varphi^2 + n\\varphi - 1 = n - 1$ 的恒等式成立（其中$\\{x\\}$表示$x$的小数部分）。这实际上是上述 Beatty 定理互补性的体现 (威佐夫游戏 - 维基百科，自由的百科全书)。

由$\\varphi$定义的数列 $W_1(n) = \\lfloor n\\varphi \\rfloor$（下 Wythoff 数列，又称为费波纳奇分割数列A，OEIS编号 A000201）和 $W_2(n) = \\lfloor n\\varphi^2 \\rfloor$（上 Wythoff 数列，又称为费波纳奇分割数列B，OEIS编号 A001950）各自严格递增，且两个数列的并集为全体正整数且互不相交 (威佐夫游戏 - 维基百科，自由的百科全书)。这两个数列在组合博弈中的经典问题——威佐夫博弈（Wythoff Nim）中出现，分别对应该博弈“冷点”（后手必胜局面）的两个坐标 (Wythoff's game - Wikipedia)。

复合数列的概念如引言所述，即对$W_1,W_2$的嵌套迭代。形式化地，对任意由1和2组成的序列$l_1,l_2,\\dots,l_k$，定义：

  • 当$k=1$时，$W_{l_1}(n)$ 即为前述$W_1(n)$或$W_2(n)$；
  • 当$k>1$时，$W_{l_1l_2\\dots l_k}(n) := W_{l_1}\\big(W_{l_2\\dots l_k}(n)\\big)$。

例如，$W_{12}(n) = W_1(W_2(n))$，$W_{212}(n) = W_2(W_1(W_2(n)))$。为方便描述，我们记 $r_i = l_1 + l_2 + \\cdots + l_i$ 为前$i$个下标之和（特别地，$r_k = l_1 + \\cdots + l_k$)。可以验证，对于上述例子有： $W_{12}(n) = W_1(n) + W_2(n)$， $W_{22}(n) = W_1(n) + 2W_2(n)$， $W_{212}(n) = W_2(W_1(W_2(n))) = W_{2(12)}(n) = 2W_1(n) + 3W_2(n)$ 等等。这些具体结果中的系数$(1,2,3,\\dots)$也出现在斐波那契数列中，提示我们可以用斐波那契数对一般的$W_{l_1\\dots l_k}(n)$进行描述。下一节将正式陈述这一一般定理并给出证明。

定理与证明

定理 1. （斐波那契系数表示） 对于任意正整数 $n$ 以及任意由 $1,2$ 组成的有限序列 $l_1,l_2,\\dots,l_k$，都有如下恒等式成立：

$$ W_{\\,l_1 l_2 \\dots l_k}(n) \\;=\\; F_{\\,r_k-2}\\,W_1(n) \\;+\\; F_{\\,r_k-1}\\,W_2(n) \\;-\\; \\sum_{i=1}^{\\,k}\\,(2 - l_i)\\,F_{\\,r_i - 1}\\,, $$

其中 $r_i = \\sum_{j=1}^i l_j$ 为前 $i$ 项下标之和。

证明： 我们采用数学归纳法证明上述公式。首先，当 $k=1$ 时，命题即断言对于$l_1=1$或$l_1=2$分别有：

  • 若 $l_1 = 1$，则 $W_1(n)$ 应满足 $$W_1(n) = F_{1-2}W_1(n) + F_{1-1}W_2(n) - (2-1)F_{1-1}\\,.$$ 由初始值 $F_{-1}=1,\\;F_0=0$ 可验证右式变为 $1\\cdot W_1(n) + 0\\cdot W_2(n) - (1)\\cdot F_0 = W_1(n)$，恰等于左侧。
  • 若 $l_1 = 2$，则 $W_2(n)$ 应满足 $$W_2(n) = F_{2-2}W_1(n) + F_{2-1}W_2(n) - (2-2)F_{2-1}\\,,$$ 右式代入$F_0=0,\\;F_1=1$后为 $0\\cdot W_1(n) + 1\\cdot W_2(n) - 0 = W_2(n)$，与左侧相符。

因此 $k=1$ 时定理成立。

接下来，假设对长度为 $k$ 的任意复合下标序列$l_1,\\dots,l_k$恒等式成立，即假设：

$$ W_{\\,l_1l_2\\dots l_k}(n) \\;=\\; F_{\\,r_k-2}\\,W_1(n) \\;+ F_{\\,r_k-1}\\,W_2(n) \\;- \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1}\\,. \\tag{IH} $$

在此基础上考虑长度为 $k+1$ 的序列$l_1,l_2,\\dots,l_k,l_{k+1}$。按照复合数列的定义，有： $$ W_{\\,l_1l_2\\dots l_k l_{k+1}}(n) \\;=\\; W_{\\,l_1l_2\\dots l_k}\\!\\Big(W_{\\,l_{k+1}}(n)\\Big)\\,. $$ 将 $X = W_{\\,l_{k+1}}(n)$ 视作新的输入，由归纳假设 (IH)，我们可以将右侧表示为： $$ W_{\\,l_1\\dots l_k l_{k+1}}(n) \\;=\\; F_{\\,r_k-2}\\,W_1(X) \\;+ F_{\\,r_k-1}\\,W_2(X) \\;- \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1}\\,, $$ 其中 $X = W_{\\,l_{k+1}}(n)$。下面我们分$l_{k+1}$的取值情况进行讨论：

  1. 若 $l_{k+1} = 1$，则 $X = W_1(n)$。此时需要对上式中的 $W_1(X)$ 和 $W_2(X)$作具体化替换，即计算 $W_1(W_1(n))$ 和 $W_2(W_1(n))$。根据之前验证的具体恒等式，当$n$为任意正整数时，有： $$W_1(W_1(n)) \\;=\\; W_{11}(n) = W_2(n) - 1,$$ $$W_2(W_1(n)) \\;=\\; W_{21}(n) = W_1(n) + W_2(n) - 1.$$ 将这两式代入，可以得到：

\\[ \\begin{aligned} W_{\\,l_1\\dots l_k 1}(n) &= F_{\\,r_k-2}\\,\\big(W_2(n) - 1\\big) \\;+ F_{\\,r_k-1}\\,\\big(W_1(n) + W_2(n) - 1\\big) \\;-\\; \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1} \\\\[6pt] &= F_{\\,r_k-2}\\,W_2(n) \\;+ F_{\\,r_k-1}\\,W_1(n) \\;+ F_{\\,r_k-1}\\,W_2(n) \\;-\\; \\Big(F_{\\,r_k-2} + F_{\\,r_k-1}\\Big) \\;-\\; \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1} \\\\[6pt] &= F_{\\,r_k-1}\\,W_1(n) \\;+ \\big(F_{\\,r_k-2} + F_{\\,r_k-1}\\big)\\,W_2(n) \\;-\\; \\big(F_{\\,r_k-2} + F_{\\,r_k-1}\\big) \\;-\\; \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1} \\\\[6pt] &= F_{\\,r_k-1}\\,W_1(n) \\;+ F_{\\,r_k}\\,W_2(n) \\;-\\; F_{\\,r_k} \\;-\\; \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1}\\,, \\end{aligned} \\] \\]

其中第三步利用了斐波那契数的递推关系 $F_{\\,r_k} = F_{\\,r_k-1} + F_{\\,r_k-2}$ 将系数合并。注意到当$l_{k+1}=1$时，$r_{k+1} = r_k + 1$，因此 $F_{\\,r_k} = F_{\\,r_{k+1}-1}$， $F_{\\,r_k-1} = F_{\\,r_{k+1}-2}$。又因为此时$(2 - l_{k+1}) = (2-1) = 1$，上式右端$-F_{\\,r_k}$可与求和中新增的一项 $-(2 - l_{k+1})F_{\\,r_{k+1}-1} = -F_{\\,r_k}$ 合并。于是，上式化简为：

$$ W_{\\,l_1\\dots l_k 1}(n) \\;=\\; F_{\\,r_{k+1}-2}\\,W_1(n) \\;+ F_{\\,r_{k+1}-1}\\,W_2(n) \\;-\\; \\sum_{i=1}^{\\,k+1} (2 - l_i)\\,F_{\\,r_i-1}\\,. $$

  1. 若 $l_{k+1} = 2$，则 $X = W_2(n)$。类似地，我们应用已知恒等式计算 $W_1(W_2(n))$ 和 $W_2(W_2(n))$： $$W_1(W_2(n)) \\;=\\; W_{12}(n) = W_1(n) + W_2(n),$$ $$W_2(W_2(n)) \\;=\\; W_{22}(n) = W_1(n) + 2\\,W_2(n).$$ 将这两式代入归纳式，得：

\\[ \\begin{aligned} W_{\\,l_1\\dots l_k 2}(n) &= F_{\\,r_k-2}\\,\\big(W_1(n) + W_2(n)\\big) \\;+ F_{\\,r_k-1}\\,\\big(W_1(n) + 2W_2(n)\\big) \\;-\\; \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1} \\\\[6pt] &= \\big(F_{\\,r_k-2} + F_{\\,r_k-1}\\big)\\,W_1(n) \\;+ \\big(F_{\\,r_k-2} + 2F_{\\,r_k-1}\\big)\\,W_2(n) \\;-\\; \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1} \\\\[6pt] &= F_{\\,r_k}\\,W_1(n) \\;+ \\big(F_{\\,r_k-1} + F_{\\,r_k}\\big)\\,W_2(n) \\;-\\; \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1} \\\\[6pt] &= F_{\\,r_k}\\,W_1(n) \\;+ F_{\\,r_k+1}\\,W_2(n) \\;-\\; \\sum_{i=1}^{\\,k} (2 - l_i)\\,F_{\\,r_i-1}\\,, \\end{aligned} \\] \\]

其中我们利用 $F_{\\,r_k-2} + 2F_{\\,r_k-1} = F_{\\,r_k+1}$ 将$W_2(n)$的系数化为斐波那契数。注意当$l_{k+1}=2$时， $r_{k+1} = r_k + 2$，因此 $F_{\\,r_k} = F_{\\,r_{k+1}-2}$， $F_{\\,r_k+1} = F_{\\,r_{k+1}-1}$。另外，此时$(2 - l_{k+1}) = (2-2) = 0$，所以原求和中从$i=1$到$k$的部分已经是新的求和到$k+1$，不需要额外增加项。由此可得：

$$ W_{\\,l_1\\dots l_k 2}(n) \\;=\\; F_{\\,r_{k+1}-2}\\,W_1(n) \\;+ F_{\\,r_{k+1}-1}\\,W_2(n) \\;-\\; \\sum_{i=1}^{\\,k+1} (2 - l_i)\\,F_{\\,r_i-1}\\,. $$

综上，在 $l_{k+1}=1$ 和 $l_{k+1}=2$ 两种情况下， $W_{\\,l_1l_2\\dots l_k l_{k+1}}(n)$ 都满足所需的恒等式形式。这说明假设 $k$ 时成立可以推得 $k+1$ 时也成立。根据数学归纳法原理，命题对一切正整数 $k$ 均为真。因此定理1得证。$\\;\\square$

结论

本文通过引入复合数列的概念，将源自黄金分割率的两个互补数列 $W_1(n)=\\lfloor n\\varphi\\rfloor$ 和 $W_2(n)=\\lfloor n\\varphi^2\\rfloor$ 与斐波那契数列建立了紧密联系。定理1给出了任意复合序列 $W_{l_1l_2\\dots l_k}(n)$ 的一个封闭形式：它可以表述为基本数列 $W_1(n)$ 和 $W_2(n)$ 的线性组合，其中系数恰为斐波那契数。这一定理统一解释了先前人们通过实验观察到的特殊算例（如 $W_{12}(n)=W_1(n)+W_2(n)$， $W_{22}(n)=W_1(n)+2W_2(n)$， $W_{222}(n)=3W_1(n)+5W_2(n)$ 等），从理论上揭示了斐波那契数在此类分割数列组合中的普适性。

值得注意的是，黄金分割率与斐波那契数列间的这种巧妙联系在其他类似问题中也有所体现。由于任意满足 $\\frac{1}{\\alpha} + \\frac{1}{\\beta} = 1$ 的无理数对$(\\alpha,\\beta)$都可生成一对互补的Beatty数列，人们或许可以考虑推广上述结论至更一般的情形。例如，当$\\alpha$是二次无理数时，$\\alpha$和$\\beta$（由 $\\frac{1}{\\alpha} + \\frac{1}{\\beta} = 1$ 确定）生成的互补数列可能与某种线性递推数列相关（黄金分割率的情况对应于斐波那契递推）。进一步的研究可以探索这样的复合数列公式在其它数字体系下是否仍然成立，以及这些结果在组合博弈论（例如Wythoff博弈的位置分析）或数字拆分问题中可能的应用。总而言之，本文的新定理丰富了人们对黄金分割序列结构的理解，为解析和应用此类数列提供了一个有力的工具。
